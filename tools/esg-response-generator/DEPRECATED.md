# DEPRECATED

This folder is superseded by **response-ready** (CY/response-ready/).

The questionnaire response engine has been extracted into a standalone, domain-agnostic package.
ESG Passport now consumes response-ready directly.

A copy of this folder is archived at `CY/Old/eu-tools-esg-rg/`.

This folder can be safely deleted once the corrupted directory names are resolved
(the `src"{components` directory has special characters that prevent normal deletion on Windows).
