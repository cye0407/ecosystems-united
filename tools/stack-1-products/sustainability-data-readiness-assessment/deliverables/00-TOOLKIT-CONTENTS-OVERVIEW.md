## File Format Standards

### Excel Files (.xlsx)

- **Compatibility:** Excel 2016 and later, Google Sheets compatible
- **Protection:** Formula cells locked, input cells unlocked (yellow background)
- **Validation:** Dropdown lists for categorical inputs, number ranges for scores
- **Automation:** Formulas auto-calculate results, charts auto-update
- **Instructions:** Every workbook has an Instructions tab as the first sheet

### Word Files (.docx)

- **Compatibility:** Word 2016 and later, Google Docs compatible
- **Structure:** Clear headings, table of contents where appropriate
- **Editability:** [CUSTOMIZE HERE] placeholders marked clearly
- **Formatting:** Consistent fonts, spacing, professional appearance

### PDF Files (.pdf)

- **Quality:** High-resolution, print-ready
- **Navigation:** Bookmarks and clickable table of contents
- **Size:** Optimized for reasonable file size (<5MB per file)
- **Accessibility:** Text selectable, not just images
